## File Format

All piece images should be in SVG format for scalability and crisp rendering at any size.

## Dimensions

- Recommended size: 64x64px
- ViewBox: 0 0 64 64
- Stroke width: 2px
- Fill: Solid colors (no gradients)

## Colors

### White Pieces
- Fill: #ffffff
- Stroke: #000000

### Black Pieces
- Fill: #000000
- Stroke: #ffffff
